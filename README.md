# Jar-OJ-frontend
Jar-OJ项目前端部分

## 根据后端接口, 快速生成前端API
```shell
openapi --input http://localhost:8121/api/v2/api-docs --output ./generated --client axios
```

## Project setup
```
yarn install
```
